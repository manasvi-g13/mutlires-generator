MutliRes: Resume Generator & Job Matcher

To run this you have to do the following:
- To clone the the repo, run `git clone https://github.com/manasvi-g13/mutlires-generator.git`
- Move into project directory: `cd mutlires-generator`
- Install `npm install`
- Create a `.env` file and then put in the following: `OPENAI_API_KEY`, `HF_TOKEN`, `JOB_TABLE`, `PGHOST`, `PGDATABASE`, `PGUSER` and `PGPASSWORD`.
- To run the project: `npm run dev`
